# Calculator
A simple calculator written in C.

## How to compile it
Run:
```
gcc src/calculator.c -o calculator
```

### Usage:
```
./calculator
```
Runs a basic calculator without parameters.

### Example:
```
./calculator -id 235 -oh
```
Expected Output:
```
Input Decimal: 235
Result in hexadecimal is: EB
```

#### Help:
With parameters
```
-v = version
-id = input decimal
-ih = input hexadecimal
-od = output decimal
-oh = output hexadecimal
```
